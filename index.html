I will be picking one page of a website, identifying problems, and then redesigning it!




Why this page?

I chose this website because I think the header and footer can be improved easily, and their design isn’t really seamless. I feel like this application page can be updated to match other recently, well designed applications.


Some problems:

Efficiency:

  • I think the submit button should be centered or on the right side.
  • I want to explore how it would look if all the options were aligned in columns rather than two in a row. That way the user can just scan vertically.

Learnability:

  • I think the website does a good job describing what to do at each question. However, for the questions with options, I plan to make them radio buttons.

Memorability:

  • The header and heading area doesn’t do a great job creating a memorable brand for the New England Aquarium. I plan to fix the header and footer areas to make the design more consistent. There are two of the same logos and one of them does not match the background color.

Accessibility Problems:

  • While using WebAIM WAIVE, a few contrast problems were detected between the text and background colors. This is for when phrases like “Select option” were written in the text box. I think it might be okay because the box is empty and it might seem intuitive to click to select the option.
  • Another problem was that the contrast was low where you could select what page you wanted to go to. I agree, this could be fixed.
  • Lastly, the website is missing a first level heading. I think this could be fixed.
  • It seems like the website is well done in terms of aria labels.
